**A work in progress project for PG3402.**

This project is a microservice based Pokemon TCG website. Users will be able to:
* Open booster packs
* Collect and manage their Pokemon
* Sell Pokemon
* Battle against trainers

**How to run:**
* cd BoosterPack
* mvn clean install
* cd User
* mvn clean install
* docker compose up
* Head to http://localhost:5173/
